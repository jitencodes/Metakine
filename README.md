### Run Locally

To run **Metakine** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/jitencodes/Metakine.git
```

Windows:

```bash
git clone https://github.com/jitencodes/Metakine.git
```
